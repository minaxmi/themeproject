Plugin Installation

The theme uses the TGM Plugin Activation script that prompts you to install required and recommended plugins after you activate the theme.

Required plugins are essential to provide the core functionality the theme relies on. You can still use the theme without required plugins, however, its features would be limitted.

With recommended plugins you get the peace of mind that the theme was tested with these plugins and even provides a design and functional integration. You can ommit their installation if you want, but we recommend installing them.

Other plugins than required and/or recommended should also work with the theme, but they might require additional tweaks to integrate them 100% with the theme. There are thousands of plugins available for WordPress and it is not possible for any theme developer to test the theme with all of them. This theme was tested with most of the popular plugins. You can install any custom plugin following the WordPress instructions (or the plugin's instructions in plugin's documentation).

To install and activate required plugin, Click "Begin Installing Plugins" from any of your admin page.

Then a list of required plugins will appeared and select all plugins, select install dropdown option and click apply.
