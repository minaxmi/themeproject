Wander Essentials Plugin Installation

The Wander Essentials Plugin a part of Wander is have separate plugin named "wander-essentials". The description below explains how to install it.

Method 1: Via The WordPress Dashboard Message "This theme requires the following plugin"

  1. Just click Begin installing plugin and Install requered pluging wander-essentials.zip and others

Method 2: Via The WordPress Dashboard (Easier)

  1. Unzip the package you downloaded from Themeforest, inside you will find a folder called Plugin and inside this you will find a file called wander-essentials.zip
  2. Login to your WordPress dashboard, go to Plugins > Add New and click on the Upload
  3. Click the Browse button and find the wander-essentials.zip file
  4. After the plugin has successfully uploaded click Activate and you're done!

Method 3: Via FTP

  1. Unzip the package you downloaded from Themeforest, inside you will find a folder called Plugin and inside this you will find a Zip File called wander-essentials.zipthan Unzip this file and you get folder called wander-essentials
  2. Login to your web server via FTP and browse to wp-content/plugins and upload the wander-essentials folder here
  3. Login to your WordPress dashboard and go to Plugins, the wander-essentials plugin will be visible, click Activate and you're done!
