Menu and front page

To set up home, firstly you need to select Onepage Menu from menu section.

Then include the menu items you needed.

Then follow the process to set Frontpage Template.

  1. Go to Page > Add Page
  2. Name the page i.e. "Frontpage"
  3. Select Template For Single Page Section From Template
  4. Published your page
  5. To setup the home page, in your WordPress admin panel, navigate to Settings > Reading.
  6. Assuming you already have a page named "Frontpage", for the Front page display, select "A static page" and choose your "Frontpage" page as the home page. This is what you should be seeing on your screen from left side.
